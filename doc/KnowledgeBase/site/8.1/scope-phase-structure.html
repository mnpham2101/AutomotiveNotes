8.1 20-Day Training Plan

Scope — Phase Structure

  • Phase 1 (Days 1–4): Process & Architecture Foundations — 1.1, 2.1, 2.2.
  • Phase 2 (Days 5–9): Platform & Embedded Fundamentals — 3.1, 3.2, 5.1.
  • Phase 3 (Days 10–14): Hands-On Worked Example — 4.1 and an attempted local build.
  • Phase 4 (Days 15–17): Lifecycle & Tooling Breadth — 6.1, 7.1, 7.2.
  • Phase 5 (Days 18–20): Leadership Practice & Synthesis — communication/team-management drills + full review.
  • Phase 6 (Days 21–28): Advanced Topics — Knowledge-Sharing Extension — suggested topics and research goals drawn from the full AUTOSAR Layered Software Architecture specification (Doc ID 53, R22-11 — far deeper than the 2.1/2.2 summaries) and the official AUTOSAR Workflow Example (Doc ID 1145), covering material this knowledge base only summarizes or doesn't yet cover at all.
